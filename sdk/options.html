LEGO SDK TOOL CONFIGView On Github

1. 插件说明

该插件服务于素材库表单和素材库物料的内容提取和设置，兼容线上线下。只要业务端对接了素材库，均可以使用该插件进行表单填写和查看复制；只要是素材库做的物料，在广告展示页面均可以使用该插件对物料内容进行查看和复制。

2. 插件使用

该插件主要用来减少测试物料时繁琐的人工操作，线上物料内容提取等，具体使用方法可咨询我，或者  查看图解

3. 插件配置

设置插件有权限的url，逗号分隔，支持换行，支持字符串部分匹配和正则匹配（全集为*.baidu.com，若有本地需求，请自行修改host映射到xx.baidu.com即可）
